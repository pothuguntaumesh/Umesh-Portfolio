  • Nesh – AI-powered chrome extension for seamless text assistance (WIP) ⌄

    Built Nesh as a Chrome extension to revolutionize how we interact with text on the web using LLMs. Simply typing /nesh in any text field invokes AI assistance, while right-clicking provides options to rewrite, improve, or generate content. The vision is to make it context-aware – able to see what you see and instantly guide you in doing things or fixing whatever you need on websites just by prompting it. Currently working to open source this project so everyone can use and help develop it, making AI assistance truly ubiquitous across the web.

    Chrome Extension API JavaScript LLM APIs Context Injection DOM Manipulation
  • Clearhold – Compliance checker for Independent financial advisors ⌄

    Created ClearHold for a friend who needed a solution to check compliance for social media posts. This tool is specifically designed for Independent Financial Advisors and Independent Insurance Agents to verify that their content adheres to regulatory compliance requirements. The platform analyzes social media posts and flags potential compliance issues before they're published, helping professionals maintain their licenses while engaging effectively with clients on social platforms.

    Note: This project was never fully completed. After building the basic functionality, my friend didn't see the potential for it to scale, so development was discontinued. Only the core compliance checking features were implemented.

    Tech Stack
    LLM APIs Reasoning Models Compliance Rules Engine Text Analysis

  • Knights tour visualizer – My first coding project journey ⌄

    This holds a special place as my very first project, built almost 5 years ago when I was learning Python from YouTube tutorials. I had just discovered backtracking algorithms and was inspired by someone's Sudoku solver video, but I wanted to create something original that wasn't already out there. The Knights Tour problem fascinated me – the challenge of moving a knight across a chessboard visiting every square exactly once. I built a real-time visualizer using Python's Turtle module that shows the knight's movements as it explores the board, backtracking when it hits dead ends. It was my first taste of bringing algorithms to life visually.

    Tech Stack
    Python Turtle Graphics Backtracking Algorithm Real-time Visualization Recursion
